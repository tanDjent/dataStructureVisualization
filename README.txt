# dataStructureVisualization

This is a prototype of a larger project of data structure visualization which will contain most of the sorting algorithms
and data structure such as linked list, stack, queue, binary search tree, etc.

Reference/Inspired by-https://www.cs.usfca.edu/~galles/visualization/AVLtree.html

https://sdkcarlos.github.io/sites/artyom.html  (Artyom JS library is used for voice recognition, it requires a https connection to function which can be estabilished locally by
generating a SSL certificate.)

I used https://www.youtube.com/watch?v=ke1SgU_HY80 as a reference to create my own SSL certificate to use on Tomcat 9.
